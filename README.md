## 项目结构说明：

```
  └─src
      └─main
          ├─java
          │  └─cn
          │      └─visiontalk
          │          └─skillfulread
          │              ├─biz                业务核心
          │              │  ├─domain          领域层
          │              │  │  ├─model        领域对象
          │              │  │  └─repository   领域仓储：封装领域对象的行为           
          │              │  ├─external        外部服务调用
          │              │  │  ├─api          封装外部服务:类命名为微服务名称
          │              │  │  └─enums        外部服务需要引用的枚举
          │              │  └─service         核心服务
          │              │      └─impl        核心服务实现
          │              ├─constant           常量
          │              ├─enums              枚举
          │              ├─facade             门面
          │              │  ├─api             对外api
          │              │  │  └─v1           对外提供服务实现(分版本)
          │              │  ├─handler         统一拦截、统一异常处理
          │              │  └─model           出参入参对象
          │              └─persistence        存放mybatis文件
          │                  ├─auto           自动生成的文件
          │                  │  ├─mapper      映射java interface
          │                  │  └─model       即DTO
          │                  └─ext            手动添加
          │                      ├─mapper     映射java interface
          │                      └─model      自定义的vo,类似数据库中view
          └─resources                         资源文件
```
